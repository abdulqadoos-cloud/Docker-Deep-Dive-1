# Docker Installation on AWS EC2

## Project Overview

This project demonstrates how to launch an Ubuntu EC2 instance on AWS and install Docker Engine.

## Technologies Used

- AWS EC2
- Ubuntu 24.04
- Docker
- Linux CLI

## Steps Performed

1. Created an EC2 Ubuntu instance.
2. Connected using SSH.
3. Updated system packages.
4. Installed Docker Engine.
5. Verified Docker installation.
6. Ran a test container.
